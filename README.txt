Personal Status Bar -- это аддон WoW, для кастомного приватного сервера Sirus, который отображает полоски здоровья и основного ресурса (маны, энергии, ярости) возле модели персонажа по центру экрана.

Функции:
    Отображение текущего здоровья и ресурса.
    Отображение значения ресурсов в процентах.
    GUI и перемещение по оси X и Y (скоро в будущих версиях)

Вызов меню через чат-команду: /psb

Параметры по умолчанию выглядят так: /psb scale 1.2 | alpha 0.8 | x 0| y -150 | percent | reset

---------------
Ключи команд      
---------------
scale = размер полосок бара.
alpha = уровень прозрачности от 0 до 2
x = положение оси Х относительно центра.
y = положение оси Y относительно центра.
percent = включить или отключить отображение процентов на полоске " /psb percent ON" или " /psb percent OFF"
reset = сбросить настройки по-умолчанию.
